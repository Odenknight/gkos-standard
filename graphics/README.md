# GKOS graphics

- `GKOS_Logo_Provisional.svg` — provisional working logo adopted under R8-072.
- `GKOS_OSI_Comparison.svg` — **v0.75-era** comparison of the OSI network model and GKOS knowledge-governance model. Retained as the historical baseline; superseded in presentation (not deleted) by the v0.76 graphics below.
- `GKOS_Poster_Corrected.png` — **v0.76** one-page summary poster (repository map, OSI/GKOS comparison, human/agent/system responsibilities, simplified knowledge flow).
- `GKOS_OSI_Comparison_3D_Corrected.png` — **v0.76** revision of the OSI/GKOS layer analogy (3D-box rendering), the direct successor to `GKOS_OSI_Comparison.svg`; also closely parallels `illustrated/figures/fig2-osi-analogy.png`.
- `GKOS_Implementation_Dashboard_Corrected.png` — **v0.76** six-panel dashboard covering the GKOS model, OKF+ layer mapping, the OSI analogy, a Kosmos-Oden v0.6.5 implementation-status map, GCP conformance profiles, and human/agent/system responsibilities.

## Version progression

`GKOS_OSI_Comparison.svg` (v0.75) → `GKOS_OSI_Comparison_3D_Corrected.png` (v0.76) documents how the OSI analogy graphic evolved between releases. Both are kept in the repository so the progression is visible rather than erasing the earlier version; the v0.75 SVG is not linked from current-state documentation but remains available as release history (added 2026-07-22, alongside the Illustrated Edition).

# GKOS graphics

- `GKOS_Logo_Provisional.svg` — provisional working logo adopted under R8-072.
- `GKOS_OSI_Comparison.svg` — modern comparison of the OSI network model and GKOS knowledge-governance model.
- `GKOS_Poster_Corrected.png` — corrected one-page summary poster (repository map, OSI/GKOS comparison, human/agent/system responsibilities, simplified knowledge flow). Covers the same comparison as `GKOS_OSI_Comparison.svg` plus repository and release orientation; kept as a separate file pending a decision on whether it supersedes the SVG.
- `GKOS_OSI_Comparison_3D_Corrected.png` — alternate 3D-box rendering of the OSI/GKOS layer analogy, closely paralleling `illustrated/figures/fig2-osi-analogy.png`.
- `GKOS_Implementation_Dashboard_Corrected.png` — six-panel dashboard covering the GKOS model, OKF+ layer mapping, the OSI analogy, a Kosmos-Oden v0.6.5 implementation-status map, GCP conformance profiles, and human/agent/system responsibilities.

The three "corrected" images above were added 2026-07-22 as new files without overwriting any existing graphic. Their relationship to `GKOS_OSI_Comparison.svg` and to the illustrated edition's figures is unresolved — they overlap in content and may be intended as replacements, alternates, or standalone additions. A maintainer should confirm intent before any existing graphic is retired.

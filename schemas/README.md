# GKOS schemas

Machine-readable schemas are planned for Source Records, Structured Knowledge Objects, assertions and lineage, control receipts, Decision Records, Context Manifests, Authorized Use Records, Agent Contracts, conformance manifests, and OKP-CH exchange packages.

v0.75 defines semantic requirements but does not yet declare a complete normative serialization. Schema proposals must preserve the master standard’s authority, provenance, temporal, sensitivity, and identity invariants.

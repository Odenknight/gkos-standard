# Release notes — GKOS-2026-07-20 v0.76

GKOS v0.76 is a concept-refinement and documentation-clarification release.

It:

- defines the v0.x series as developmental testing and refinement rather than formal consensus ratification;
- freezes the twelve-state epistemic vocabulary directly in GKOS;
- defines four previously ambiguous mechanical terms;
- adds full progressive-disclosure requirements;
- publishes informative layer-artifact and authority-receipt annexes;
- separates implementation version claims from GKOS release claims;
- adds a dedicated R9 Development Decision Record;
- publishes the GKOS Engine and Specialist Reviewer designs as standalone informative references.

The release does not complete the authority-receipt mechanism, actor-identity model, upward attestation chain, decision-record integrity scheme, executable conformance suite, second implementation, signed publication, DOI, or v1.0 consensus governance.

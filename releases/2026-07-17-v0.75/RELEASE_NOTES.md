# Release notes — GKOS-2026-07-17 v0.75

Initial public release of the Governed Knowledge Operations Standard.

The release includes the seven-layer reference model, authority and epistemic rules, layer interface contracts, Specialized Agent Framework, provisional GCP profiles, Viewer/Projection Profile, security/privacy/retention requirements, governance, dual licensing, contribution DCO, graphics, known limitations, and decisions R1-001 through R8-075.

The executable GKOS-TS suite, final OKP-CH schemas, independent implementation evidence, certification process, signed manifest, DOI, and v1.0 governance are not complete.

# GKOS fixture catalog

The fixture program will contain positive, negative, boundary, contradiction, sensitivity, delegation, replay, erasure, context-reproduction, and authorized-use cases for GCP-1 through GCP-7 and the Viewer/Projection Profile.

Fixtures are incomplete in v0.75. A conformance claim must disclose which fixtures were executed, versions, outcomes, exceptions, and whether evaluation was self-attested or independent.

# Code of Conduct

GKOS participation requires evidence-focused, respectful professional conduct.

Expected behavior includes criticizing proposals rather than people, disclosing conflicts, preserving minority technical positions, distinguishing fact from inference, and respecting privacy and security boundaries.

Unacceptable behavior includes harassment, threats, discrimination, coercion, fabricated evidence, doxxing, deliberate disruption, and disclosure of confidential vulnerability information.

Report public conduct concerns through repository maintainers or GitHub platform reporting. Do not include sensitive personal data in a public issue. Enforcement may include warning, content removal, temporary restriction, or permanent exclusion, with a recorded rationale where disclosure is safe and lawful.

# Minimal governed lifecycle

1. Preserve a source revision and issue a Source Record.
2. Create a Structured Knowledge Object with a stable UID.
3. Record an assertion linked to exact evidence.
4. Run mandatory validation and issue control receipts.
5. Submit a proposal and record an authorized decision.
6. Compile a purpose-bound Context Manifest including contradictions and warnings.
7. Perform an authorized action and issue an Authorized Use Record.
8. If accepted knowledge changes, trace dependent uses and issue compensation events where required.

# GKOS conformance

GKOS defines cumulative provisional profiles GCP-1 through GCP-7 plus a Viewer/Projection Profile. A higher profile inherits all applicable lower requirements.

Claims must include a machine-readable manifest, human-readable report, standard and test versions, evidence, limitations, and exceptions. Self-attested and independently verified claims must be distinguished. v0.75 provides provisional requirements; the executable GKOS-TS suite is incomplete.

# Annex — Conformance profiles

GCP profiles are cumulative capabilities aligned with the seven GKOS layer contracts.

- GCP-1: Source preservation
- GCP-2: Structure and stable identity
- GCP-3: Typed assertions and lineage
- GCP-4: Validation and deterministic control
- GCP-5: Governed review and Decision Records
- GCP-6: Reproducible Context Manifests
- GCP-7: Authorized Use Records

The Viewer/Projection Profile is read-only and must display provenance, epistemic state, incompleteness, contradictions, warnings, and conformance limitations.

Claims MUST identify the exact standard release, profile, assessment type, evidence, exceptions, and test-suite status.
